Hello, world

This site — and now this blog — is hand-written HTML, CSS and JavaScript, hosted free on GitHub Pages. No frameworks, no build step: what you read is exactly what sits in my editor.

I plan to write here about what I build — Flutter apps, SQL pipelines, the occasional hand-rolled encryption engine — and about what breaks along the way, because that is where the actual learning lives.

What a post can contain

A paragraph is just a <p> tag. Inline code like flutter run uses <code>. Longer code goes in a <pre> block: 

-- from the credit-risk pipeline: default rate by grade
SELECT grade,
       AVG(default_flag) AS default_rate
FROM loans
GROUP BY grade
ORDER BY grade;

  • Bullet points reuse the site's arrow style.
  • Good for step-by-steps and takeaways.

Credit Risk & Loan Default Analytics

End-to-end credit-risk pipeline on 2.26M Lending Club loans — from raw CSV to a defended approval-cutoff decision.

SQL feature engineering in DuckDB (window functions, vintage curves, segment risk tables), calibrated gradient-boosting PD models in Python with strict leakage discipline, a 4-page executive Power BI dashboard, and a risk-committee memo that defends the cutoff in business terms — expected credit loss, not model accuracy.

The finding: grade F–G, debt-consolidation, low-income borrowers default at 50–58% — 2.9× the book average.

The Cipher Vault

A 100% offline encrypted journal, task list and focus timer — with an AES-256 engine written by hand in C++.

  • Note bodies encrypted by a hand-written C++ AES-256 engine called from Dart over FFI — verified against the NIST FIPS-197 test vector. Plain text never touches disk.
  • PIN gate stores only a SHA-256 hash; the key is derived in memory and never persisted.
  • Deep Work mode: flip the phone face-down and the accelerometer switches to a dark focus theme and starts a Pomodoro session.
  • No cloud, no network, no telemetry.
